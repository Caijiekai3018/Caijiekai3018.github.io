Elements in HTML: Video

Hypertext Markup Language Label

Path: html > p01 > 02

<video> is a label to load the video in the web.
You can use it to play the video in the web to improve the feelings on the web.
Here is a test about the <video> label.

The forms of the video support are: MP4, OGG, WebM.

Support Form
Edge Safari Firefox Opera Chrome
MP4 Yes Yes Yes(From v21) Yes(From v25) Yes
OGG Yes No Yes Yes Yes
WebM Yes No Yes Yes Yes

The grammar of this label:
<video src=" " width=" " loop=" " autoplay=" " controls=" " height=" " poster=" " preload=" ">
src: The addr of the source.
width: The width of the video.
height: The height of the video.
autoplay: Play the video when the web load.
controls: Displays the video controls.
poster: Specifies the image that is displayed while the video is being downloaded until the user clicks the play button.
preload: If this property appears, the video loads when the page loads and is ready to play. If "autoplay" is used, this property is ignored.
loop: If present, playback starts again when the media file finishes playing.
