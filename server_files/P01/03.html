Elements in HTML: Image

Hypertext Markup Language Label

Path: html > p01 > 03

The HTML5 <img> tag is an essential element for displaying images on a web page. It provides a simple and straightforward way to embed images into HTML documents. The <img> tag has several attributes that allow you to control various aspects of the image, such as its source, size, alt text, and more.

The most important attribute of the <img> tag is the src attribute, which specifies the URL of the image file. You can provide a relative or absolute URL to the image file, depending on its location. For example:

<img src="image.jpg" alt="A beautiful image">

In the above example, the src attribute specifies the relative URL of the image file "image.jpg". The alt attribute provides alternative text for the image, which is displayed if the image fails to load or for accessibility purposes.

The <img> tag also allows you to control the size of the image using the width and height attributes. These attributes specify the width and height of the image in pixels. For example:

<img src="image.jpg" alt="A beautiful image" width="500" height="300">

In the above example, the image will be displayed with a width of 500 pixels and a height of 300 pixels. It's important to note that specifying the dimensions of the image using these attributes can result in distorted images if the aspect ratio is not maintained.

To maintain the aspect ratio of the image, you can specify either the width or the height attribute and let the browser automatically calculate the other dimension. For example:

<img src="image.jpg" alt="A beautiful image" width="500">

In the above example, the image will be displayed with a width of 500 pixels, and the browser will automatically calculate the height to maintain the original aspect ratio.

Furthermore, the <img> tag supports the alt attribute, which provides alternative text for the image. This text is displayed if the image fails to load or for users who rely on screen readers. It's important to provide descriptive and meaningful alternative text to ensure accessibility and improve the user experience.

In addition to the basic attributes, the <img> tag also supports various other attributes, such as title for adding a tooltip to the image, loading for controlling the image loading behavior, decoding for specifying the image decoding method, and more. These attributes provide additional control and customization options for the image element.

In conclusion, the HTML5 <img> tag is a fundamental element for displaying images on the web. It allows you to embed images into HTML documents easily and provides various attributes to control the image source, size, alternative text, and more. By utilizing the <img> tag effectively, you can enhance the visual appeal and accessibility of your web pages.
